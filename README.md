# Jacob Zang

做 BSP 和底层系统开发，平时主要和板子、内核、驱动、启动链路、系统裁剪这些东西打交道。

我做过 Amlogic、Rockchip 和 NVIDIA Jetson 相关平台的项目，工作里经常需要把硬件、系统、应用之间的问题串起来看：有时候是一个启动异常，有时候是一个驱动问题，有时候是系统集成后某个很难复现的边角 bug。

这个博客主要用来放一些我自己的记录。不是教程合集，也不是包装简历，更多是把实际做项目时遇到的问题、排查过程和一些阶段性的理解写下来。很多东西当时不记，过一阵就只剩下“我好像处理过”，所以这里也算是给自己留一份可回头看的笔记。

## 主要方向

- BSP bring-up、系统移植和板级适配
- Linux kernel、设备树、驱动调试
- U-Boot、启动流程和量产相关问题
- Amlogic、Rockchip、Jetson 平台实践
- 嵌入式 Linux 系统裁剪、构建和集成
- 项目里遇到的问题复盘和排查思路

## 最近会整理的内容

- 不同平台 BSP 开发里的共性问题
- 内核和设备树调试记录
- 系统启动、升级、量产相关的坑
- 一些嵌入式项目里的工程化经验

## 关于我

我对“把问题定位清楚”这件事比较感兴趣。底层开发里很多问题不是单点原因，日志、原理图、内核配置、设备树、驱动代码、构建脚本都可能有关。写博客也是想把这些零散经验整理成更完整的线索，方便自己以后查，也希望能帮到遇到类似问题的人。
